# **App Name**: Executive View

## Core Features:

- Calendar Integration: Fetch and display travel (Feb 9-12) and personal trips (Mar 25-29 to Singapore) from Google Calendar using the Calendar API.
- Workload Visualization: Generate a chart showing team member workload, differentiating between standard projects and urgent ('boss order') tasks.
- Assignments List: Maintain a separate, sortable list for tasks assigned by the executive, complete with a 'Sync to Google Keep/Tasks' button.
- Project Status Integration: Aggregate status updates for NextGen ERP and Mobile CRM projects into the dashboard.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to evoke a sense of authority and strategic insight.
- Background color: Light gray (#F0F2F5) to provide a clean, neutral backdrop.
- Accent color: Teal (#009688) for highlighting key data points and actionable items.
- Body and headline font: 'Inter', a sans-serif font providing a modern, clean and neutral reading experience.
- Use minimalist, line-style icons to represent data categories and actions, ensuring clarity and avoiding visual clutter.
- Employ a card-based layout with clear sections for each data source. Cards should be well-spaced to aid quick scanning and comprehension.
- Subtle transitions and animations when data updates or filters are applied, providing visual feedback without being distracting.
